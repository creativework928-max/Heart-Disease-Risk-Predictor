# ❤️ Heart Disease Risk Predictor

## 🚀 End-to-End Machine Learning Deployment Project

An interactive web application that leverages Machine Learning to assess the probability of heart disease based on patient clinical and demographic information. This project demonstrates the complete machine learning lifecycle, from data preprocessing and feature engineering to model training, evaluation, and deployment using Streamlit.

---

## 📌 Project Overview

Cardiovascular disease remains one of the leading causes of death worldwide. Early risk assessment can help identify high-risk individuals and support preventive healthcare decisions.

This application allows users to enter patient health metrics and instantly receive:

* Heart disease risk prediction
* Probability-based risk assessment
* Risk categorization (Low, Moderate, High)
* Interactive visualizations
* Clinical recommendations based on predicted risk

The project was developed as a comprehensive Machine Learning deployment solution showcasing production-ready practices and modern data science workflows.

## 📂 Dataset

Dataset Source:

**UCI Heart Disease Dataset**

The dataset contains patient health indicators commonly used for cardiovascular disease assessment, including:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol Level
* Fasting Blood Sugar
* Resting ECG Results
* Maximum Heart Rate
* Exercise-Induced Angina
* ST Depression (Oldpeak)
* Number of Major Vessels
* Thalassemia Information
